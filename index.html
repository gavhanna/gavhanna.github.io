Projects

DONT PANIC eCommerce Store

Live App Github Link

An eCommerce store built for a Server Side Web Tech class project in NCI

It allows users to view store items, add and remove cart items and finalise purchases.
Users can view a profile page with some information on their account including an order history.
Admin users can grant and remove admin status to other users, add edit and delete store items.

It uses nested components that interact with each other via services and rxjs observables. The app itself is hosted on Amazon Web Services. Firebase is used as the project's backend.

The app utilises a PostgresSQL database and is deployed on Heroku.

Dublin Bikes App

Live App Github Link

A Dublin Bikes app built in one weekend.

It allows users to view a google map of each Dublin Bike Station location, including an interactive Streetview of each selected location. Users can save locations to a favourite list for quick viewing of availablity and location status.

This app uses Google's Firebase as a backend, handling both authentication and data storage. I chose to use Bulma css to quickly create the UI.

Simon Game

Live App Github Link

A Simon Game App built in vanilla JS.

It plays an incrementing number of coloured lights, accompanied by tones. The user has to remember and input the same sequence. If the user fails, the sequence is repeated. If playing on "strict mode", the game resets on a failed attempt. The game ends after 20 rounds.

This app was built as part of the Free Code Camp's front end projects. It is hoted on github pages and was built using vanilla JS and CSS.

ISS Tracker

Live App Github Link

An app that allows users to track the position of the Internation Space Station.

Users can click on a menu button for some extra information, current velocity, altiture and visibility. The green circle indicates the best view of the ISS as it passes.

This app was built in a few hours one Saturday for fun. It uses Google Maps API and wheretheiss developer API. It uses jQuery to implement the AJAX requests.
